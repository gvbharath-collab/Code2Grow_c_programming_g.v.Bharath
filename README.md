day 15 -- students performance analyser

main.c is the main source code file

day15.exe is executable file

This challenge requires designing a program that analyzes the performance of students in a class. The program takes marks of up to 10 students, calculates the total and average marks, identifies the highest and lowest scores, counts passed and failed students, and provides a summary of grade distribution (A+, A, B, C, D, Fail) using arrays and conditional logic.
