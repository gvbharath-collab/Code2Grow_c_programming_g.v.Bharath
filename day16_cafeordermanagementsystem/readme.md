This program manages customer orders in a café, calculates each bill, total revenue, and identifies the most and least ordered items.

Main Variables:

prices[5] – stores prices of 5 items.

quantitiesSold[5] – tracks how many of each item are sold.

numCustomers, numItems, itemNumber, quantity – handle customer inputs.

totalRevenue, totalItemsSold – store overall café totals.

Working Logic:

Displays a fixed menu with prices.

Asks for the total number of customers (max 10).

For each customer:

Takes the number of items ordered.

Repeatedly asks for item number and quantity.

Calculates the bill and updates totals.

After all customers, it finds which item was ordered the most and least.
